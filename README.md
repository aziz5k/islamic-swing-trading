# نظام التداول الإسلامي — Islamic Swing Trading System

نظام متكامل لإدارة استراتيجية **Swing Trading** الإسلامية للسوق الأمريكي (NYSE / NASDAQ)

## 🖥️ لقطة شاشة
واجهة عربية RTL كاملة مع لوحة تحكم، قائمة مراقبة، نظام نقاط، وسجل صفقات.

---

## ✨ الميزات

| الصفحة | الوصف |
|--------|-------|
| 📊 لوحة التحكم | إحصائيات حية، نسبة الفوز، ربح/خسارة، التقدم نحو الهدف |
| 👁 قائمة المراقبة | 13 سهماً مع Musaffa / Zoya + إضافة وحذف |
| 🎯 نظام التقييم | حساب النقاط 100/100 قبل كل صفقة مع قرار فوري |
| 💼 سجل الصفقات | فتح/إغلاق الصفقات مع حساب الربح/الخسارة تلقائياً |
| ✅ قائمة التحقق | 19 بنداً يومياً مع نتيجة فورية |
| 📜 القواعد الذهبية | العشر قواعد + دليل VIX + مسار الصفقة |
| 💰 رأس المال | تتبع النمو التراكمي والتوقعات السنوية |

---

## 🚀 التشغيل

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل السيرفر
npm start

# افتح المتصفح على:
http://localhost:3000
```

## 📐 قواعد الاستراتيجية

| البند | القيمة |
|-------|--------|
| رأس المال | $25,000 |
| حجم كل صفقة | $1,000 – $2,500 |
| الهدف الأول | +5% → بيع 50% |
| الهدف الثاني | +8% → بيع الباقي |
| وقف الخسارة | -7% ثابت |
| الحد الأقصى للصفقات | 4 صفقات متزامنة |

## 🛠️ التقنيات

- **Backend**: Node.js + Express.js
- **Database**: SQLite (better-sqlite3)
- **Frontend**: HTML / CSS / JavaScript — واجهة عربية RTL
- **تصميم**: Dark theme احترافي
